# Framework
```
The project is based on Chrome 104 Beta and ChromeDriver 104 due to the existing Chrome 103 bug during project creation.
https://bugs.chromium.org/p/chromedriver/issues/detail?id=4121

The path to the Chrome 104 Beta executable must be specified in the 'wdio.conf.js'.
```
# Installation
```
npm install
```

# Run automated scripts
```
npm run test-all-chrome
npm run test-smoke-chrome
npm run test-others-chrome
npm run test-all-firefox
npm run test-smoke-firefox
npm run test-others-firefox
```
